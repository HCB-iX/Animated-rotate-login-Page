# Animated-rotate-login-Page
An interactive, 3D login page where the login card smoothly rotates 360 degrees to reveal the signup form when clicked. It replaces static page loads with fluid, hardware-accelerated motion to keep users engaged.

---

## Features

- **Pure HTML5 & CSS3:** No external JS libraries or frameworks required.
- **Dynamic Circular Animation:** Built using CSS Custom Properties (`--i`) and `calc()` to arrange and animate a 360-degree rotating light ring.
- **Floating Placeholders:** Interactive input labels that smoothly shrink and elevate when a user focuses on an input field.
- **Cyberpunk / Dark Aesthetics:** Styled with a sleek dark palette (`#1f293a`) and high-contrast neon cyan (`#0ef`) accents.
- **Google Fonts Integration:** Utilizes the clean modern typography of the **Poppins** font family.
